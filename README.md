# Simple Platformer Game

A fun browser-based platformer game where you navigate through increasingly challenging levels!

## How to Play

1. **Controls:**
   - **Arrow Keys Left/Right**: Move left and right
   - **Space Bar or Arrow Up**: Jump
   
2. **Objectives:**
   - Jump across platforms to reach the green goal flag
   - Collect golden coins for bonus points (+10 each)
   - Avoid red enemies
   - Don't fall off the bottom!

3. **Scoring:**
   - Collect coins: +10 points each
   - Complete a level: +50 points bonus
   
4. **Levels:**
   - 3 levels of increasing difficulty
   - Each level has more platforms, enemies, and coins

## Files

- `index.html` - Main game page
- `styles.css` - Game styling and animations
- `game.js` - Game logic and mechanics

## Getting Started

1. Open `index.html` in your web browser
2. Use arrow keys and space bar to control your character
3. Reach the green goal flag to advance to the next level
4. Try to beat all 3 levels with the highest score!

## Features

- Smooth physics and collision detection
- Multiple levels with different layouts
- Enemy AI with patrol behavior
- Collectible coins for bonus points
- Score and level tracking
- Game Over and Win screens
